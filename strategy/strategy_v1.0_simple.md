## 1. 💵 现金储备桶（~$1,250）

**作用**：卫星分批建仓的弹药 + 抄回调 + 心理缓冲。**它不是闲钱,是"等机会的子弹"。**

**3 条规则**：
1. 卫星桶"分 3 次买"的第 2、3 笔,钱从这里出。
2. **地板 $1,000,不许跌破**——除非出现过了卫星 5 条准入的真机会;"追热门"冲动**不算**机会。
3. 赌场亏了,**绝不**从现金/核心补血。

---

## 2. ⚓ 核心桶（~$1,250）

**作用**：账户的压舱石,也是**治"踏空"的主药**——你永远在市场里,就不会因为"没参与"而焦虑。

**规则**：
- 标的：**只买 SPY 或 QQQ**(宽基,别买主题 ETF)。
- 方式：**定投买入持有**。比如每周 / 每两周固定金额买一点,不择时。
- **不设止损、不做波段、不交易。** 这个桶的纪律就是"什么都不做"。
- 跌了不慌——宽基长期向上,跌了反而是定投的好时候。

## 3. 🛰️ 卫星桶（~$4,160）= 固定结构 + 准入清单

> **核心理念：名单随市场轮动(什么强买什么),但每只票进来前必须过同一道关。** 这样"随市场变"不会退化成"追热门"。

### 3.1 固定结构（写死,不变）

- **最多 4 个槽位**(4 只票)。
- **单只 ≤ 20% NAV(~$1,500)。**
- **同一主题 ≤ 2 只**;且**至少 1 只非 AI/科技**(消费/医药/能源/金融…真正分散)。
- **分 3 次建仓**:想买先买 1/3,不猜回调;确认或回调再补 2/3(钱从现金桶出)。
- **买入即挂止损**:下买单那一刻,止损单一起挂上。
- **止盈有规则**(治"拿不住",二选一,固定用一种)：
  - (a) **移动止损法**:涨上去后把止损往上移(锁住利润,让趋势跑);或
  - (b) **分批减法**:每涨一档(如 +20%/+40%)减 1/3,留底仓。
  - ❌ 不许"小赚就清仓"凭感觉跑。

### 3.2 准入清单（想往任何槽位放一只票,5 条必须全过）

1. ☑️ 能写出**一行"为什么拿它几个月"的逻辑**（不是"它在涨"）。
2. ☑️ **中大盘、流动性好**（不碰低价股 / 仙股 / 单股 2X 杠杆 ETF）。
3. ☑️ **当场能定出止损位**（invalidation：跌到哪我就认错走人）。
4. ☑️ 放进去**不破"同主题 ≤2 / 至少 1 只非 AI"**。
5. ☑️ **看到它涨,过了 24h 还想买**（冷静期,杀冲动）。

> **过不了这 5 条的票 → 不许进卫星,只能进赌场桶(且受 $650 封顶约束)。**
> SpaceX 这类 IPO 首日 / 纯热门追涨,大概率过不了第 1、2、5 条 → 自动落到赌场。

## 4. 🎰 赌场桶（$650,硬封顶）

**作用**：给"怕踏空 / 追热门 / 想冲 IPO / 想玩期权"的冲动一个**有界的出口**。把冲动隔离,而不是压抑——压抑会让它跑去污染卫星桶。

**规则**：
1. **总额硬封顶 $650(绝不超 $800)。** 亏光为止,**绝不补血**。
2. 所有**博主推荐 / 新闻热点 / IPO 首日 / 期权 / 低价股**,只能在这里玩。
3. 单笔别太大,别让一次冲动用光整个桶。
4. 想冲之前,问自己一句:"这笔过得了卫星 5 条准入吗?"过得了就走卫星流程;过不了,认清它就是赌博,愿赌服输。

## 5. 日常工作流（极简版,替代旧的复杂流程）

**每天（几分钟）**：
- 扫一眼持仓和挂单是否正常(止损单还在不在)。
- **不需要**频繁操作。没有过准入的新机会,就什么都不做。

**想买一只新票时**：
1. 走**卫星 5 条准入清单**。
2. 过了 → 分 3 次建仓,买入即挂止损,记一行逻辑 + 止损位。
3. 没过 → 进赌场桶(查是否还在 $650 额度内),或放进观察清单等 24h。

**看到博主/新闻心动时**：
- **24h 内不准下单。** 写进观察清单,明天再看还想不想买。

**亏损时**（关键,因为你会想逃避）：
- 止损单已经事先挂好了 → **让规则替你执行,你不用临场做决定**。
- ❌ 不许对没设止损的票"加仓摊低成本等回本"(MU 这种高位票尤其)。

## 6. 红线清单（违反任何一条 = 停手复盘）

- [ ] 现金桶 < $1,000
- [ ] 赌场桶 > $800
- [ ] 卫星单只 > 20% NAV
- [ ] 卫星同主题 > 2 只 / 全是 AI、无分散票
- [ ] 卫星某只票**没有**挂止损
- [ ] 看到博主/新闻**当天**就下单(没冷静 24h)
- [ ] 持有 <3 个交易日就平仓(day-trade)——每周统计次数,目标 ≤1

## 7. 这套策略怎么对症（画像 → 解法对照）

| 你的毛病 | 这套策略怎么治 |
|---|---|
| 怕踏空 | 核心桶让你永远在场 + 赌场桶给冲动出口 |
| 不会定 timing | 分 3 次建仓,不用猜回调 |
| 拿不住盈利 | 移动止损 / 分批减,规则化止盈 |
| 亏了不看盘 | 买入即挂止损,自动保护 |
| 跟博主冲动 | 24h 冷静 + 5 条准入 + 赌场封顶 |
| 板块集中 | 同主题 ≤2 + 至少 1 只非 AI |
| 高频乱动 | 极简流程 + day-trade 计数红线 |
