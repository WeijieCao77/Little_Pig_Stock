## 策略框架定位

`strategy/framework_v0.6.md` 是参考性的"北极星"，**非硬性规则**。
框架核心价值：
- PM 画像（中大盘 + 行业龙头 + risk-averse + 单笔 15-25% 收益目标）
- 阶段 A Pre-event Run-up 完整六步流程
- v0.6 决策树（财报性质判定 → Step 2A/2B/2C）
- "不做"哲学：硬阈值过滤模糊机会

每次分析都用框架作为参考，但具体决策考虑当前实战环境。

## 使用方法

每次重大调仓、入金、或宏观事件（FOMC、关税、特朗普重要发言）后，
新建一份 `analysis/YYYY-MM-DD_xxx.md` 记录决策依据。这样可以回头看
自己当时的逻辑是否成立，是新手最重要的训练。
