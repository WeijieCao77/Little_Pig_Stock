## 2. 目标

- **年化 ~25%**（激进,但在合理上沿;**前提是改掉"追涨循环"**）

## 3. 风险画像（自述 vs 真实——必须诚实）

| 维度 | 旧 framework 写的 | 真实的用户（实证） |
|---|---|---|
| 标签 | risk-averse、不投机 | **行为上相当激进/投机**：2X 杠杆 ETF、低价股+期权双押、追涨 |
| 收益心态 | 单笔吃 15-25% 即满意 | **拿不住,小赚就跑**;同时又**怕踏空想吃大行情** |
| 真实约束 | — | **"避免大回撤 / 不归零" > "博最大收益"**（因为怕归零、亏了会逃避） |
| 心理止损线 | 账户 -15~20% | **可能更低**——"亏多了就不想看股票"说明真实痛感阈值偏低 |

**结论**：用户的真实风险画像是**"嘴上保守、手上激进、心里怕大跌"**的矛盾体。策略要做的不是压制激进(压不住),而是**给激进一个有边界的出口**(赌场桶),同时**用结构保护住大头**(核心+现金)。

---

## 4. 心理结构（策略设计的核心依据）

1. **🔑 最痛 = 踏空 > 亏损。** 看着没买的票一路涨,比自己持仓亏钱更难受。**这是所有追涨行为的总根源。**
2. **入场障碍**：想等回调买,但**不会定 timing**——猜不准回调多少能进,错过后忍不住追高。
3. **持有障碍**：**拿不住盈利的票**（小赚就跑,吃不到大段）。
4. **亏损反应不一致**（用户原话归纳）：
   - 没设止损 + 意外下跌 → **慌了割肉 或 逃避不看**
   - 有信念的票(如 MU) → **加仓摊低成本,等回本** ⚠️ 对高位票是危险习惯
   - 事先计划好的止损 → **能正常按计划出局** ✅
5. **易被带动**：博主 / 推文 / 新闻 → **冲动下单**。
6. **盯盘悖论**：平时能每隔几分钟看一次,**但大亏时回避**——**风控最该在的时候,人却不在场。**

## 6. 行为实证（6 周交易得出,不是猜测）

- **模式**：高频翻仓(1-6 天)、板块过度集中(半导体/光通信曾占 36-38%)、卖了又追(ASTS/MU)、churn 送摩擦成本(LITE2X 一周来回 6 趟)。
- **盈亏归因**（精确,见 `2026-05-29_pnl_breakdown` 数据）：
  - 赚钱靠**少数拿住的票**：MU +$271、ASTS +$196、BE +$111、MEM +$74、QCOM +$110
  - 亏钱靠**手痒乱动**：CIEN −$151(反复来回搓)、SATS −$47、CRWV −$38、ORCL −$35,以及一堆 1-6 天就割的计划外仓
- **铁律级结论**：**用户赚钱靠"拿住",亏钱靠"乱动"。** 策略的全部目标就是放大前者、掐死后者。

## 7. 关键错配（为什么旧策略不适用）

- **framework v0.8 是用户一位有金融背景的友人(金融数学研究生)做的专业事件驱动框架**,反映的是一个专家的打法,**不是为用户设计的**。
- 用户是**新手 + 怕踏空 + 时间有限 + 亏了逃避**,与 v0.8 的「财报/事件驱动 + A/B/C 三阶段决策树 + 做空梯子(SQQQ/Put)」**方向性不匹配**。
- 用户自评 v0.8 里"事件驱动大方向 / 三阶段太复杂 / 做空用不上 / 硬规矩太束缚"——**几乎整套都不适配**。
- **处理**：v0.8 不删,移到 `strategy/archive/` 作为**进阶参考 + 学习教材**,等用户成熟后回看。日常用为用户量身定制的**极简策略 v1.0**(见 `strategy/strategy_v1.0_simple.md`)。

## 8. 画像 → 策略设计约束（直接推导）

| 用户的问题 | 策略必须给的结构性解法 |
|---|---|
| 怕踏空 | **核心仓常驻市场**(SPY/QQQ),你永远"在场",踏空焦虑自然降 |
| 不会定入场 timing | **机械分 3 次建仓**,不用猜回调幅度 |
| 拿不住盈利 | **规则化止盈/移动止损**,写死,不靠当时判断 |
| 亏了不看盘 | **买入即挂止损**(自动保护,不靠临场盯盘) |
| 跟博主/新闻冲动 | **观察清单 + 24h 冷静 + 准入清单**;过不了的只能进赌场桶 |
| 激进压不住 | **赌场桶**给冲动一个 ≤$650 的有界出口,亏光不伤本 |
| 新手 + 小本金 | **极简 + 低频**(规则少到能背;少交易省摩擦) |

## 9. 复查节奏

- 本画像每 **1-2 个月**或账户/生活发生重大变化(大额加减仓、加钱、目标变)时复查一次。
- 下次复查重点观察项：**"持有 <3 个交易日就平仓"的次数(day-trade 频率)** 是否下降、**板块集中度**是否守住、**赌场桶**有没有越界。
